## A06. 점수 계산
국어, 영어, 수학 점수를 입력받아 이 점수의 총점과 평균을 계산하여 출력하라. 
>총점 = 국어점수 + 영어점수 + 수학점수    
평균 = 총점 / 3.0 

변수는 다음과 같이 사용하라.
>int kor; // 국어점수    
int eng; // 영어점수    
int math; // 수학점수    
int total; // 총점    
double average; // 평균점수



#### * 실행예시
<pre><code>Enter points of Korean > 80
Enter points of English > 85
Enter points of Math > 95
Total point is 260
The average is 86.7
</code></pre>
