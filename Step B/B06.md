## B06. 점수 계산

국어, 영어, 수학 점수를 입력받아 이 점수의 총점과 평균을 계산하고, 각 과목별로 90점 이상이면 성적우수 로 표시하여 출력하라.
>총점 = 국어점수 + 영어점수 + 수학점수    
평균 = 총점 / 3.0 

변수는 다음과 같이 사용하라.
>int kor, eng, math; //국어점수, 영어점수, 수학점수    
int total; // 총점    
double average; // 평균점수
